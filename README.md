# Coffee Shop Website

A clean, fully responsive website built for a coffee shop client. Features a full menu showcase, gallery, about section, and contact form — designed for both mobile and desktop.

## Live Preview

Open `index.html` in any browser — no server required.

## Features

- Fully responsive layout (mobile, tablet, desktop)
- Menu section with categories (Hot Drinks, Cold Drinks, Food)
- Photo gallery with hover effects
- About us and story section
- Contact form with validation
- Smooth scroll navigation
- Google Maps embed for location

## File Structure

```
├── index.html          # Main page
├── Css/
│   └── style.css       # All styles
├── Assets/
│   ├── images/         # Product and hero images
│   └── icons/
├── Pages/
│   ├── menu.html
│   └── contact.html
└── .vscode/
```

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (Flexbox, Grid, animations)
- JavaScript (form validation, smooth scroll)

## Customization

Edit `Css/style.css` to change colors/fonts. Replace images in `Assets/images/` with your own.
